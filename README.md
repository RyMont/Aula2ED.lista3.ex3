# Exercício 3 — Inversão de Cadeia de Caracteres (Recursivo)

Criar uma função recursiva que exiba o resultado da inversão de uma cadeia de caracteres.

Exemplo:  
Entrada = teste  
Saída = etset

Deve-se utilizar a função SUBSTRING da Java.

O código deve trazer como comentários:
- A condição de parada;
- Como escrever a função para o termo *n* em função do termo anterior.
